# Infinity Army Builder

A personal army builder for the Infinity N4 tabletop game by Corvus Belli.
Built with Flask (Python) and vanilla JavaScript.

## ⚠️ Data & Assets
This project does not include game data or images.
All Infinity data, artwork and assets are property of Corvus Belli S.L.
This tool is for personal, non-commercial use only.

## Setup
1. Add your own `data/` folder with Infinity JSON data
2. Add your own `static/images/` folder
3. `pip install flask`
4. `python app.py`

## Features
- Filter units by skills, equipment and weapons
- Track owned miniatures
- Army list builder
